# FE-CLI

## Edit

  ```shell
  # development
  npm run dev
  ```

 OR

  ```shell
  # production
  npm run prod
  ```

## Use

  ```shell
  npm install
  npm link
  fe init <name>
  # example
  fe init zhy
  # help
  fe init --help
  ```

## Format
  
  ```shell
  npm run format
  ```

## Test
  
  ```shell
  npm run test
  ```
